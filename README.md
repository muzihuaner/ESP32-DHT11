# 🌿 ESP32 温湿度监控系统（网页版 + 飞书推送）

基于 ESP32 和 DHT11 传感器的本地温湿度监控系统，支持**独立网页实时查看**与**飞书机器人定时推送**。  
网页界面现代美观，自动显示网络同步的北京时间，数据每 10 秒刷新一次。

---

## 📋 功能特点

- ✅ 实时采集温度（℃）与湿度（%）
- ✅ 本地网页可视化展示，自适应手机/电脑
- ✅ 网页展示**网络同步北京时间**，数据时效清晰
- ✅ 每 5 分钟自动推送一次环境报告到飞书群机器人
- ✅ 纯离线本地服务器，不依赖第三方云平台
- ✅ 代码简洁，易于二次开发

---

## 🔧 硬件需求

| 硬件 | 型号/说明 | 数量 |
|------|------------|------|
| 主控 | ESP32 开发板 | 1 |
| 温湿度传感器 | DHT11（或 DHT22，需修改代码）| 1 |
| 连接线 | 面包板 + 杜邦线 | 若干 |
| 电源 | Type-C 数据线 | 1 |

---

## 📦 软件依赖（Arduino 库）

在 Arduino IDE 中安装以下库：

- **DHT sensor library** by Adafruit （版本 ≥ 1.4.4）
- **ArduinoJson** by Benoit Blanchon （版本 ≥ 6.0）
- ESP32 开发板包（自带 WiFi、HTTPClient、time 等基础库）

> 安装方法：`工具` → `管理库` → 搜索库名并安装。

---

## 🌐 硬件连接

| DHT11 引脚 | ESP32 引脚 |
|------------|------------|
| VCC（正极）| 3.3V 或 5V |
| DATA（数据）| GPIO 4（可自行修改代码）|
| GND（负极）| GND |

> 如果 DHT11 模块带有上拉电阻，接线更简单。若使用裸传感器，需在 DATA 与 VCC 之间连接一个 4.7kΩ~10kΩ 电阻。
